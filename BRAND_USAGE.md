# AgentPalette 品牌使用说明

本说明适用于 `AgentPalette` 名称、Logo、应用图标和其他可识别为官方产品的品牌标识（统称“品牌标识”）。品牌标识不包含在 [Apache License 2.0](./LICENSE) 对代码的授权中。

## 允许的使用

- 为准确说明本项目、引用本项目或链接至官方仓库而使用名称。
- 在未经修改的官方安装包、官方 Release 或官方仓库镜像中保留品牌标识。
- 在说明兼容性或来源时，以事实准确且不造成混淆的方式使用名称。

## 不允许的使用

- 将 Fork、修改版或第三方构建称为官方 AgentPalette，或暗示获得官方认可。
- 在衍生版本、分发包、域名、社交账号或应用商店条目中使用足以造成来源混淆的名称或 Logo。
- 使用品牌标识推广与本项目无关的产品或服务。

## Fork 与衍生版本

你可以在 Apache-2.0 的范围内 Fork、修改和分发代码，但应使用清晰不同的项目名称、应用图标和发布渠道，并在适当位置说明其基于 AgentPalette 构建。

如需超出上述范围使用品牌标识，请先联系仓库维护者取得书面许可。
